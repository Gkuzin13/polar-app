## Installation

#### Prerequisites

- npm

1. Clone the repo
   ```sh
   git clone git@github.com:Gkuzin13/polar-app.git
   ```
2. Install npm packages
   ```sh
   npm install
   ```
3. Start the development server

   ```sh
   npm start
   ```

   In browser, open http://localhost:3000
